GitHub

Como criar uma conta

O GitHub é uma plataforma que lhe permite a criação um de conta gratuita, para isto basta
seguir o passa-a-passo a seguir:
1- Acesse o link: https://github.com/

2- Já na página do Git você deverá clicar em "Inscreva-se no Github" ou
"Sign up for GitHub", em inglês.

3- Logo depois deverá preencher alguns campos:
3.1- Informando seu e-mail;
3.2- Criando uma senha;
3.3- Inserindo um nome de usuário;
3.4- Informando se gostaria de receber atualizações e anúncios de produtos por e-mail;
3.5- Resolver alguns enigmas, simples, de verificação;
3.6- E por fim clique em "Criar conta" ou "Create account", em inglês.

4- O GitHub irá enviar um código de lançamento para o e-mail cadastrado, basta inseri-lo.

5- Pronto, sua conta está criada, agora basta desfrutar de todas as ferramentas do GutHub.

Como criar um repositório

O GitHub permite a criação de repositórios para armazenamentos de dados pessoais,
não necessariamente um código, veja a seguir como é simples a criação de um repositório:
1- Abra o GitHub em https://github.com/ e faça login com a sua conta já criada.

2- Na página inicial clique no botão "Novo repositório" ou "New repository", em inglês.

3- Nomeie seu repositório, e caso deseje, adicione uma descrição sobre o mesmo.

4- Pronto, seu repositório está criado, agora basta usá-lo como desejar.
