O que é Git?

Para você que também fica perdido em questão de saber a diferença entre Git e Git-Hub, fique calmo, todos os iniciantes assim como nós passamos por essa dúvida e tentaremos esclarecer essa dúvida de uma vez por todas.

O Git é um sistema de controle de versão open-source, ou seja, ele é gratuito. Ele é utilizado para a criação de um histórico de alterações em código-fonte de projetos de desenvolvimento de software.

Foi desenvolvido por Linus Torvalds, o criador do sistema operacional Linux. Por meio de sua utilização, podemos saber quais foram as alterações realizadas, quem fez cada uma das alterações e baixar essas mudanças em nossa máquina. Se necessário, revertê-las para uma versão anterior.
